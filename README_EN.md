# Izanami-Lab

Muv-Luv Girls Garden X Battle Simulator.

> This is a fan-made tool and is not affiliated with âge / aNCHOR / Anchor.

## Features

- Global parameter configuration (school levels, equipment, character levels, rarity, modules, affection, skills)
- Character parameter override system
- Custom dummy enemy creation
- Team formation (2x3 grid) and battle simulation
- Step-by-step crit simulation
- Tactical exercise mode
- Preset management
- Dark/light/system theme support
- Batch simulation with statistics
- Battle log export

## How to Run

### Option 1: Python Environment

```bash
pip install -r requirements.txt
python gui_app.py
```

### Option 2: Standalone EXE (No Python Required)

Download `Izanami-Lab_vX.X.X.zip` from [Releases](https://github.com/izanami-lab/izanami-lab/releases), extract, and run `Izanami-Lab.exe`.
